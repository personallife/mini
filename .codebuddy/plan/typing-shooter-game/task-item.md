- [ ] 5. 实现游戏核心场景 — 玩家飞机与敌机生成

   - 实现敌机生成器：按当前关卡配置，定时从屏幕顶部随机水平位置生成敌机
   - 敌机上方显示对应字母/单词文字标注，敌机沿直线匀速向玩家飞机方向移动
   - 实现同屏敌机数量上限控制
   - 显示 HUD 信息：得分、连击数、生命值（心形图标）、关卡进度（已击落/目标数）

- [ ] 6. 实现打字输入与射击机制
   - 实现键盘输入监听（仅字母键 a-z）
   - 实现**自由模式**：按键后匹配距离玩家最近的同字母敌机，发射子弹飞向目标
   - 实现**锁定模式**：首字母匹配后锁定敌机（高亮边框标识），依次输入剩余字母，全部正确后击落；输入错误给出视觉反馈但不解除锁定；锁定目标被击落或飞出后自动解除
   - 实现子弹飞行动画：从玩家飞机位置向目标敌机直线飞行
   - 实现子弹命中后的爆炸粒子效果
   - 根据难度级别自动设置默认输入模式（初级=自由，中高级=锁定）
   - _需求：1.3、1.4、5.1、5.2、5.3、5.4、5.5、8.2、8.3_

   - 实现最后1条生命时的危险警告效果（屏幕边缘变红闪烁）
