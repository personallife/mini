# ✈ 打字射击游戏 - Typing Shooter

一款面向儿童的打字练习游戏，融合经典"小蜜蜂"射击玩法。通过键盘输入正确的字母或单词来发射子弹击落敌机，在趣味游戏中练习打字能力！

## 🎮 在线游玩

访问 [https://personallife.github.io/mini/typing-shooter/](https://personallife.github.io/mini/typing-shooter/) 即可直接在浏览器中游玩。

## 🎯 游戏玩法

- 屏幕底部是你的飞机，顶部不断飞来带字母/单词的敌机
- 输入正确的字母/单词即可发射子弹击落敌机
- 敌机撞到你的飞机会扣血，血量归零则游戏结束
- 连续击落敌机可获得连击加分

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| A-Z | 输入字母，击落对应敌机 |
| ESC | 暂停/继续游戏 |

## 📊 难度系统

- **初级 (Easy)**: 单字母，适合打字入门 (5小级)
- **中级 (Medium)**: 短单词 3-4 字母 (5小级)
- **高级 (Hard)**: 长单词 5+ 字母 (5小级)
- **自定义模式**: 自由调节速度、频率、难度等参数

## 📚 内置词库

- 常用儿童英语单词 (200+)
- 小学课本词汇 (按 1-6 年级分类，400+)
- 支持自定义导入

## 🖥 本地运行

无需安装任何依赖，直接用浏览器打开即可：

```bash
# 方式1: 直接打开
open typing-shooter/index.html

# 方式2: 使用本地服务器（推荐）
cd typing-shooter
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🛠 技术栈

- **Phaser.js 3** - HTML5 2D 游戏框架
- **Web Audio API** - 音效合成（无需外部音频文件）
- **纯前端** - 无服务器依赖，GitHub Pages 托管
