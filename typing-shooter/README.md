# ✈ 键盘大作战 - Typing Shooter

一款面向儿童的打字练习射击游戏，融合经典"小蜜蜂"射击玩法。通过键盘输入正确的字母或单词来发射子弹击落敌机，在趣味游戏中练习打字能力！

## 🎮 在线游玩

访问 [https://personallife.github.io/mini/typing-shooter/](https://personallife.github.io/mini/typing-shooter/) 即可直接在浏览器中游玩。

## 🎯 游戏玩法

- 屏幕底部是你的飞机，顶部不断飞来带字母/单词的敌机
- 输入正确的字母/单词即可发射子弹击落敌机
- 敌机撞到你的飞机会扣血，血量归零则游戏结束
- 连续击落敌机可获得连击加分（10连击/20连击额外奖励！）

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| A-Z | 输入字母/单词，击落对应敌机 |
| ESC | 暂停/继续游戏 |

## 📊 难度系统

3大级别 × 20关 = **60关**，难度由浅入深

- **初级 (Easy)**: 单字母，适合打字入门（20关）
- **中级 (Medium)**: 短单词 3-4 字母（20关）
- **高级 (Hard)**: 长单词 5+ 字母（20关）
- **无尽模式**: 自由调节速度、频率、难度等参数，挑战最高分

## 👑 Boss 战

每5关迎战一个 **Boss**，共12个Boss！

- Boss战不会出现普通敌机，专注与Boss对决
- Boss会**发射子弹**攻击玩家，输入对应字母可抵消
- Boss受伤时偶尔掉落**回血**和**护盾**道具
- Boss血量 < 50% 进入**狂暴模式**（攻击加速）
- Boss血量 < 25% 进入**绝境模式**（更多子弹！）
- 击败Boss获得**大量奖励分数**

## 🎁 道具系统

击落敌机有概率掉落7种道具：

| 道具 | 效果 |
|------|------|
| 🛡 护盾 | 10秒内免疫一次伤害 |
| 🐢 减速 | 10秒内所有敌机减速至25% |
| 💣 炸弹 | 瞬间清除屏幕上所有敌机 |
| ❄️ 冰冻 | 5秒内冻结所有敌机 |
| ⭐ 双倍 | 8秒内得分翻倍 |
| 💚 治疗 | 恢复1点生命值 |
| 🧲 磁铁 | 8秒内道具自动飞向玩家 |

## 🎨 主题切换

支持3种场景主题：🌌 星空、🌊 海洋、🌲 森林

## 📚 内置词库

- 常用儿童英语单词 (200+)
- 小学课本词汇 (按 1-6 年级分类，400+)
- 支持自定义词库导入导出

## 💾 存档系统

- 3个独立存档槽
- 支持导出/导入存档（JSON格式）
- 自动保存关卡进度和最高分

## 🖥 本地运行

无需安装任何依赖，直接用浏览器打开即可：

```bash
# 方式1: 直接打开
open typing-shooter/index.html

# 方式2: 使用本地服务器（推荐）
cd typing-shooter
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 🛠 技术栈

- **Phaser.js 3** - HTML5 2D 游戏框架
- **Web Audio API** - 音效合成（无需外部音频文件）
- **纯前端** - 无服务器依赖，GitHub Pages 托管
